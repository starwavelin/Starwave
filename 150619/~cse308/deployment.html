  1. Instructions for installing, configuring, and starting the server side of the system:
    • Required Softwares:
      1. Windows XP/Vista/7
      2. jdk6.0
      3. NetBeans 6.9 or above with glassfish server
      4. MySQL server
    • Configures:
      1. install jdk (ex:..\java\jdkxxx);
      2. add java to environment variable; JAVA_HOME=..\java\jdkxxx path=%JAVA_HOME%\bin;
      3. NetBeans ide should recognize the java bin path, so install jdk before install NetBeans;
      4. Install MySQL(ex:..\MySQL)
        1. next-custom-next-ok-next-install
        2. create account
        3. configure mysql server
          • detailed configuration
          • developer machine
          • multi-functional database
          • choose innoDB location
          • manual setting, set max number (any reasonable number)
          • tcp/ip, set default 3306 should be fine
          • choose "multi language" will support more
          • choose a service name, default 'MySQL'
          • check launch the MySQL server automatically
          • choose a root password for yourself; do not create an anonymous account for the safety issue
          • execute, should see service started successfully
          • finish
      5. add MySQL to the environment variable
        • path = ...\MySQL\bin
      6. Test:
        1. cmd /* start command line window */
        2. java -version /* if jdk configures correctly, version will be displayed */
        3. mysql --version /* if mysql configures correctly, version will be displayed */
      7. create database:
        • mysql -u root -p
        • password
        • create database rumast;
      8. import database:
        • after log into mysql, use mysql> source d:\er.sql /*location of your sql file*/
    • Start Server:
      1. start NetBeans;
      2. open project -> select project directory;
      3. right click to set rum project as main project;
      4. run project, it will start the system default browser and display index page;
      5. if you know your ip address (ipconfig), and you have access to the router, you may set port forward to allow outside Internet access.
  2. Instructions for installing, configuring, adn starting the client side of the system:
    • Required Software:
      • IE/Firefox/Chrome or other major browsers
    • Configures:
    • LAN access if in the same LAN with Server;
    • Internet access if not in the same LAN with Server;
    • No need any extra configuration, default configure of browsers is fine.
    • Start System:
      1. open browser;
      2. enter url of the server page;
      3. start surfing.
  3. Setup for the log file
    1. Setup log file root location:
      • In this case the path for the log file to be stored is d:/log_rum/
      • Create such a folder called "log_rum" under D drive. Otherwise, an error "Problems with creating the log files" will occur.
    2. Generate a new log file:
      • Level.FINE is the minimum level that will be logged.
      • There are several standard error in the level of fine /.../info/server/warning will be logged.
      • log_register.txt will be created when new student registers.
      • log_regular.txt will be created when an actor logs in.
      • If you log more than one time on the same txt file, log will be appended/rewritten.
      • You can clear up the log history manually.
    3. Message from log file:
      • Users are able to see the message from the system. The status of the action they have performed and the error message form the system.
      • When system has message labeled as server and warning, that indicates something goes wrong.
  4. Setup SQL connection URL and path
    1. Our current database is running on a local machine.
    2. In order to work on your machine, you need to create a database on your own machines as well.
    3. Inside the DB/GoSQL.java, you will see we set up our MySQL as follows:
      • private static final String dbURL = 'jdbc:mysql://localhost:3306/rumast?';
        • The URL address and the table name as "rumast"
      • private static final String dbUser = 'root';
        • The user name as "root"
      • private static final String dbPwd = '1234';
        • the user password as "1234"
    4. Also, since we are running the database locally, you also need to create table.
      • The DB/sql.sql contains the query to create table.
      • The DB/sampleData.sql contains the query to add data into table.
